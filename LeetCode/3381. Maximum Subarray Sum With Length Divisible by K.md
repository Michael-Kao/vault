# [3381. Maximum Subarray Sum With Length Divisible by K](https://leetcode.com/problems/maximum-subarray-sum-with-length-divisible-by-k/description/)  

+ 日期 : 2025/11/27  

## 問題描述  

給定一個陣列 `nums` 和整數 `k`，找出子陣列的長度是 `k` 的倍數且和最大的子陣列和。  

## 直覺想法  

1. 先計算從 `index i` 開始，長度為 `k` 的倍數的子陣列和，並建立新的陣列  
2. 對於新陣列中的每個值，可以加入到 `i%k` 的 `prefix sum` 之中  
3. 每次 `prefix[i]` 可以是 `prefix[i]+nums[i]` 或是 `nums[i]`，取較大值  
4. 最後回傳 `prefix` 陣列中的最大值  

## 題解  

```cpp=
class Solution {
public:
    long long maxSubarraySum(vector<int>& nums, int k) {
        int l = 0;
        vector<long long> arr;
        long long cur = 0;
        long long minVal = 0;
        for(int r = 0; r < nums.size(); ++r) {
            cur += nums[r];
            if(r - l + 1 == k) {
                minVal = min(minVal, cur);
                arr.push_back(cur);
                cur -= nums[l];
                l += 1;
            }
        }
        int n = arr.size();
        long long res = LONG_MIN;
        vector<long long> prefix(k);
        for(int i = 0; i < n; ++i) {
            cur = prefix[i%k] + arr[i];
            prefix[i%k] = max(cur, arr[i]);
            res = max(res, prefix[i%k]);
        }
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
