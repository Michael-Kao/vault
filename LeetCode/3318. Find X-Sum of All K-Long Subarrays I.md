# [3318. Find X-Sum of All K-Long Subarrays I](https://leetcode.com/problems/find-x-sum-of-all-k-long-subarrays-i/description/)  

+ 日期 : 2025/11/04  

## 問題描述  

給定一個數字陣列 `nums` 和兩個整數 `k` 和 `x`，請先取出長度為 `k` 的所有子陣列，  
計算每個子陣列中出現元素的次數，並取出前 `x` 個出現次數最高的元素，  
將其相加後儲存到結果陣列中。  

## 直覺想法  

1. 使用 `hash map` 計算目前子陣列中每個元素的出現次數。  
2. 利用 `two pointer` 技巧將目前子陣列長度控制在 `k`  
3. 每次移動子陣列時，更新 `hash map` 中的元素出現次數。  
4. 當長度為 `k` 時，利用 `max heap` 取出前 `x` 個出現次數最高的元素，並將其相加後加入結果陣列。  
5. 當長度為 `k+1` 時，將左邊的元素從 `hash map` 中移除，並繼續移動右指標。  
6. 如果最後的子陣列長度小於 `k`，則依舊將子陣列中取出前 `x` 個出現次數最高的元素，並將其相加後加入結果陣列。  

## 題解  

```cpp=
class Solution {
public:
    vector<int> findXSum(vector<int>& nums, int k, int x) {
        int n = nums.size();
        unordered_map<int, int> mp;
        vector<int> res;
        int l = 0;
        for(int i = 0; i < n; ++i) {
            mp[nums[i]] += 1;
            if(i - l + 1 > k) {
                mp[nums[l]] -= 1;
                l += 1;
            }
            if(i - l + 1 == k) {
                res.push_back(getSum(mp, x));
            }
        }
        if(n - l < k)
            res.push_back(getSum(mp, x));
        return res;
    }
    
    int getSum(unordered_map<int, int> &mp, int x) {
        priority_queue<pair<int, int>> maxHeap;
        for(auto [num, cnt] : mp) {
            maxHeap.push({cnt, num});
        }
        int sum = 0;
        while(!maxHeap.empty() && x > 0) {
            auto [cnt, num] = maxHeap.top();
            maxHeap.pop();
            sum += cnt * num;
            x -= 1;
        }
        return sum;
    }
};
```

Time Complexity : $O(nklogk)$  
Space Complexity : $O(n)$
