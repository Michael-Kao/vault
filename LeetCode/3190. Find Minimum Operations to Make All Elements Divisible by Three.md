# [3190. Find Minimum Operations to Make All Elements Divisible by Three](https://leetcode.com/problems/find-minimum-operations-to-make-all-elements-divisible-by-three/description/)  

+ 日期 : 2025/11/22  

## 問題描述  

給定一個數字陣列 `nums`，可以針對 `nums` 中的元素做以下操作:  

+ 將元素加 `1` 或者將元素 `-1`  

問最少多少次操作，可以將陣列中所有元素都變成 `3` 的倍數  

## 直覺想法  

觀察非 `3` 的倍數的元素，可以分成兩類:  

1. `x % 3 == 1`: 這類可以透過一次 `-1` 操作變成 `3` 的倍數  
2. `x % 3 == 2`: 這類可以透過一次 `+1` 操作變成 `3` 的倍數  

剩餘的元素本身就是 `3` 的倍數，不需要操作，因此本題步驟如下:  

1. 遍歷陣列  
2. 如果元素不為 `3` 的倍數，則結果加 `1`  

## 題解  

```cpp=
class Solution {
public:
    int minimumOperations(vector<int>& nums) {
        int res = 0;
        for(int num : nums) {
            if(num % 3 != 0)
                res += 1;
        }
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(1)$
