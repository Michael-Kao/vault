# [1015. Smallest Integer Divisible by K](https://leetcode.com/problems/smallest-integer-divisible-by-k/description/)  

+ 日期 : 2025/11/25  

## 問題描述  

給定一個整數 `k`，問最小的正整數 `n`，其中 `n` 是由數字 `1` 組成，且 `n` 能被 `k` 整除。  

## 直覺想法  

跟昨天問題一樣，因為位數過大，所以要用餘數來解決。  

1. 起始為 `0`，使用 `hash map` 紀錄目前餘數有沒有出現過  
2. 每次更新時，將 `cur * 10 + 1`，並對 `k` 取餘數  
3. 如果餘數出現在 `hash map` 之中，表示經過一個輪迴，可以跳出迴圈  
4. 檢查 `cur` 是否為 `0`，如果是的話，回傳目前的長度 `length`，否則回傳 `-1`  

## 題解  

```cpp=
class Solution {
public:
    int smallestRepunitDivByK(int k) {
        vector<bool> seen(k);
        int res = 0;
        int cur = 0;
        while(!seen[cur]) {
            seen[cur] = true;
            cur = (cur * 10 + 1) % k;
            res += 1;
        }
        return (cur == 0) ? res : -1;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
