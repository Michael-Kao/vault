# [944. Delete Columns to Make Sorted](https://leetcode.com/problems/delete-columns-to-make-sorted/description/)  

+ 日期 : 2025/12/20  

## 問題描述  

給定一組字串陣列 `strs`，其中每個字串長度相同，將字串形成一個矩形，  
以每個 `column` 為單位，如果字元沒有照字母序排列，則刪除該 `column`，  
回傳需要刪除的 `column` 數量。  

## 直覺想法  

1. 以字串長度為外層迴圈  
2. 以陣列長度為內層迴圈  
3. 如果 `strs[i][j] < strs[i-1][j]`，則該 `column` 不符合條件，計數器加一並跳出內層迴圈  

## 題解  

```cpp=
class Solution {
public:
    int minDeletionSize(vector<string>& strs) {
        int res = 0;
        for(int j = 0; j < strs[0].size(); ++j) {
            for(int i = 1; i < strs.size(); ++i) {
                if(strs[i][j] < strs[i-1][j]) {
                    res += 1;
                    break;
                }
            }
        }
        return res;
    }
};
```

Time Complexity : $O(n*m)$  
Space Complexity : $O(1)$
