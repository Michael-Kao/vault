# [717. 1-bit and 2-bit Characters](https://leetcode.com/problems/1-bit-and-2-bit-characters/description/)  

+ 日期 : 2025/11/18  

## 問題描述  

給定一個數字陣列 `bits`，當 `bits[i] = 1`，則 `2 bits` 為一組，當 `bits[i] = 0`，則 `1 bit` 為一組。  
請判斷最後一個 `0` 是否為 `1 bit` 組成的。  

## 直覺想法  

1. 從頭遍歷 `bits`  
2. 遇到 `1` 則跳過下一個 `bit`，遇到 `0` 則繼續往下走  
3. 判斷最後一個 `0` 是否被跳過  

## 題解  

```cpp=
class Solution {
public:
    bool isOneBitCharacter(vector<int>& bits) {
        int n = bits.size();
        for(int i = 0; i < n; ++i) {
            if(bits[i] == 1) {
                i += 1;
                if(i == n-1)  return false;
            }
        }
        return true;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(1)$
