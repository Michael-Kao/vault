# [3047. Find the Largest Area of Square Inside Two Rectangles](https://leetcode.com/problems/find-the-largest-area-of-square-inside-two-rectangles/description/)  

+ 日期 : 2026/01/17  

## 問題描述  

找出兩個矩形重疊部分中，可以放入的最大正方形面積。  

## 直覺想法  

1. 計算任意兩個矩形的重疊區域  
2. 重疊區域的寬為 `min(topRightX) - max(bottomLeftX)`，高為 `min(topRightY) - max(bottomLeftY)`  
3. 當這兩個值皆大於 0 時，表示有重疊區域，則最大正方形邊長為 `min(寬, 高)`，面積為邊長的平方  

## 題解  

```cpp=
class Solution {
public:
    long long largestSquareArea(vector<vector<int>>& bottomLeft,
                                vector<vector<int>>& topRight) {
        int n = bottomLeft.size();
        int maxSide = 0;

        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                int w = min(topRight[i][0], topRight[j][0]) -
                        max(bottomLeft[i][0], bottomLeft[j][0]);
                int h = min(topRight[i][1], topRight[j][1]) -
                        max(bottomLeft[i][1], bottomLeft[j][1]);

                maxSide = max(maxSide, min(w, h));
            }
        }

        return 1LL * maxSide * maxSide;
    }
};
```

Time Complexity : $O(n^2)$  
Space Complexity : $O(1)$
