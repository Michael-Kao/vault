# [3010. Divide an Array Into Subarrays With Minimum Cost I](https://leetcode.com/problems/divide-an-array-into-subarrays-with-minimum-cost-i/description/)  

+ 日期 : 2026/02/01  

## 問題描述  

## 直覺想法  

1. `nums[0]` 是固定的，因此 `res` 初始值為 `nums[0]`  
2. 移除 `nums[0]` 後將 `nums` 排序，找出最小的兩個即可  

## 題解  

```cpp=
class Solution {
public:
    int minimumCost(vector<int>& nums) {
        int res = nums[0];
        nums.erase(nums.begin());
        sort(nums.begin(), nums.end());
        return res + nums[0] + nums[1];
    }
};
```

Time Complexity : $O(nlogn)$  
Space Complexity : $O(1)$
