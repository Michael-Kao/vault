# [1461. Check If a String Contains All Binary Codes of Size K](https://leetcode.com/problems/check-if-a-string-contains-all-binary-codes-of-size-k/description/)  

+ 日期 : 2026/02/23  

## 問題描述  

## 直覺想法  

1. 找出 `s` 能組合出長度為 `k` 的字串集  
2. 判斷是否為 `2^k` 個  

## 題解  

```cpp=
class Solution {
public:
    bool hasAllCodes(string s, int k) {
        int n = s.size();
        int left = 0;
        int right = 0;
        unordered_set<string> seen;
        while(right < n) {
            if(right - left + 1 == k) {
                seen.insert(s.substr(left, k));
                left += 1;
            }
            right += 1;
        }
        return seen.size() == pow(2, k);
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
