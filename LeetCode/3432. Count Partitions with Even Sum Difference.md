# [3432. Count Partitions with Even Sum Difference](https://leetcode.com/problems/count-partitions-with-even-sum-difference/description/)  

+ 日期 : 2025/12/05  

## 問題描述  

給定一個數字陣列 `nums`，將陣列分成兩個子陣列，計算兩個子陣列的和的差為偶數的分割方式數量。  

## 直覺想法  

1. 計算總和  
2. 擴充左子陣列的同時，減少總和，即可得到右子陣列的和  
3. 判斷兩個子陣列和的差是否為偶數  

## 題解  

```cpp=
class Solution {
public:
    int countPartitions(vector<int>& nums) {
        int sum = 0;
        for(int num : nums)
            sum += num;
        int left = 0;
        int res = 0;
        for(int i = 0; i < nums.size()-1; ++i) {
            left += nums[i];
            sum -= nums[i];
            if(!((sum - left) & 1))
                res += 1;
        }
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(1)$
