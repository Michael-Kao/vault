# [1351. Count Negative Numbers in a Sorted Matrix](https://leetcode.com/problems/count-negative-numbers-in-a-sorted-matrix/description/)  

+ 日期 : 2025/12/28  

## 問題描述  

給定一個排序過的 `m x n` 矩陣，找出矩陣中負數的數量  

## 直覺想法  

1. 遍歷矩陣  

## 題解  

```cpp=
class Solution {
public:
    int countNegatives(vector<vector<int>>& grid) {
        int m = grid.size();
        int n = grid[0].size();
        int res = 0;
        for(int i = 0; i < m; ++i) {
            int l = 0;
            int r = n-1;
            int idx = n;
            while(l <= r) {
                int mid = (r - l) / 2 + l;
                if(grid[i][mid] < 0) {
                    idx = mid;
                    r = mid - 1;
                } else {
                    l = mid + 1;
                }
            }
            res += (n - idx);
        }
        return res;
    }
};
```

Time Complexity : $O(mlogn)$  
Space Complexity : $O(1)$
