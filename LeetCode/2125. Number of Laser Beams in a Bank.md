# [2125. Number of Laser Beams in a Bank](https://leetcode.com/problems/number-of-laser-beams-in-a-bank/description/)  

+ 日期 : 2025/10/27  

## 問題描述  

給定一個字串陣列 `bank`，該字串陣列表示為一個二維二元矩陣，  
其中 `bank[i][j] = 1` 表示在第 `i` 行第 `j` 列有一個安全裝置，  
不同 `row` 之間的安全裝置會發射雷射束，問有多少雷射束會被發射出來。  

## 直覺想法  

1. 計算當前 `row` 有多少個安全裝置  
2. 如果當前 `row` 有安全裝置，則表示能與前一個有安全裝置的 `row` 發射雷射束，可發射數量為 `cur * pre`  
3. 更新 `pre` 為 `cur`  
4. 找到下一個有安全裝置的 `row` 重複步驟 1~3，直到遍歷完整個 `bank`  

## 題解  

```cpp=
class Solution {
public:
    int numberOfBeams(vector<string>& bank) {
        int m = bank.size();
        int n = bank[0].size();
        int pre = 0;
        int res = 0;
        for(int i = 0; i < m; ++i) {
            int cnt = 0;
            for(int j = 0; j < n; ++j) {
                if(bank[i][j] == '1')
                    cnt += 1;
            }
            res += cnt * pre;
            if(cnt != 0)
                pre = cnt;
        }
        return res;
    }
};
```

Time Complexity : $O(mn)$  
Space Complexity : $O(1)$
