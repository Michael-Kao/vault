# [1925. Count Square Sum Triples](https://leetcode.com/problems/count-square-sum-triples/description/)  

+ 日期 : 2025/12/08  

## 問題描述  

給定一個數字 `n`，請從 `1` 到 `n` 中找出滿足 $a^2 + b^2 = c^2$ 的組合數量。  

## 直覺想法  

1. 模擬，用兩層迴圈模擬 `a` 和 `b`，如果 `sqrt(a^2 + b^2)` 是整數且小於等於 `n`，則 `res += 2` (`a` 和 `b` 可以交換位置)  

## 題解  

```cpp=
class Solution {
public:
    int countTriples(int n) {
        int res = 0;
        for(int i = 1; i <= n; ++i) {
            for(int j = i+1; j <= n; ++j) {
                int c_square = i*i + j*j;
                int c = sqrt(c_square);
                if(c*c != c_square) continue;
                if(c <= n)  res += 2;
            }
        }
        return res;
    }
};
```

Time Complexity : $O(n^2)$  
Space Complexity : $O(1)$
