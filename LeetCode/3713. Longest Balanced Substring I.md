# [3713. Longest Balanced Substring I](https://leetcode.com/problems/longest-balanced-substring-i/description/)  

+ 日期 : 2026/02/12  

## 問題描述  

一個字串被稱為平衡，表示所有在字串中出現的字母出現次數都相同，  
找出平衡字串的最長長度  

## 直覺想法  

1. 建立所有可能子字串  
2. 檢查每個子字串是否平衡  

## 題解  

```cpp=
class Solution {
public:
    int longestBalanced(string s) {
        int n = s.size();
        int res = 0;
        for(int i = 0; i < n; ++i) {
            vector<int> cnt(26, 0);
            for(int j = i; j < n; ++j) {
                cnt[s[j]-'a'] += 1;
                bool balanced = true;
                int target = 0;
                for(int k = 0; k < 26; ++k) {
                    if(cnt[k] != 0) {
                        if(target == 0)
                            target = cnt[k];
                        if(cnt[k] != target) {
                            balanced = false;
                            break;
                        }
                    }
                }
                if(balanced) {
                    res = max(res, j-i+1);
                }
            }
        }
        return res;
    }
};
```

Time Complexity : $O(26*n^2)$  
Space Complexity : $O(26)$
