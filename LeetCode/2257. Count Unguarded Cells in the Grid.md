# [2257. Count Unguarded Cells in the Grid](https://leetcode.com/problems/count-unguarded-cells-in-the-grid/description/)  

+ 日期 : 2025/11/02  

## 問題描述  

給定兩個整數 `m` 和 `n` 以及兩個二維陣列 `guards` 和 `walls`，表示在 `m x n` 矩陣中，  
`gruads` 和 `walls` 分別在矩陣之中，並且 `guards` 可以往四個方向（上下左右）移動，  
直到遇到牆壁或另一個守衛為止，問沒有被守衛看守的格子數量。  

## 直覺想法  

1. 先將牆壁和守衛標記好  
2. 遍歷所有守衛，每個守衛去模擬走四個方向  
3. 如果遇到新的守衛或牆壁就停止，否則標記為被看守  
4. 計算矩陣中沒有被看守的格子數量  

## 題解  

```cpp=
class Solution {
public:
    int countUnguarded(int m, int n, vector<vector<int>>& guards, vector<vector<int>>& walls) {
        vector<vector<int>> seen(m, vector<int>(n, 0));
        vector<vector<int>> dir = {
            {1, 0},
            {-1, 0},
            {0, 1},
            {0, -1}
        };
        for(vector<int> wall : walls) {
            seen[wall[0]][wall[1]] = 2;
        }

        for(vector<int> guard : guards) {
            seen[guard[0]][guard[1]] = 2;
        }

        for(vector<int> guard : guards) {
            int r = guard[0];
            int c = guard[1];

            for(int i = 0; i < 4; ++i) {
                int nr = r + dir[i][0];
                int nc = c + dir[i][1];

                while(nr >= 0 && nr < m && nc >= 0 && nc < n && seen[nr][nc] != 2) {
                    seen[nr][nc] = 1;
                    nr += dir[i][0];
                    nc += dir[i][1];
                }
            }
        }
        int res = 0;
        for(int i = 0; i < m; ++i) {
            for(int j = 0; j < n; ++j) {
                if(seen[i][j] == 0)
                    res += 1;
            }
        }
        return res;
    }
};
```

Time Complexity : $O(mn)$  
Space Complexity : $O(mn)$

這題最低能的是，要事先把 `guard` 標記好，不然會 `TLE`
