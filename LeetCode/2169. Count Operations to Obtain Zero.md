# [2169. Count Operations to Obtain Zero](https://leetcode.com/problems/count-operations-to-obtain-zero/description/)  

+ 日期 : 2025/11/09  

## 問題描述  

給定兩個整數 `num1` 和 `num2`，每次操作中，選擇較大的一方扣除較小的一方，直到其中一個數字變為零。請返回所需的操作次數。  

## 直覺想法  

當 `num1 >= num2`，那麼可以進行的操作次數為 `num1 // num2`，  
接著 `num1 -= num2 * times`，隨後進行交換，直到其中一個數字為零。  

更進一步拆解，`num1 -= num2 * times` 等同於 `num1 = num1 % num2`，  
而操作完 `%` 一方肯定比較小，下一次必定是對 `num2` 進行運算，因此可用 `swap` 替換兩數，  
來減少程式碼的 `if else` 結構。  

## 題解  

```cpp=
class Solution {
public:
    int countOperations(int num1, int num2) {
        int res = 0;
        while(num1 > 0 && num2 > 0) {
            if(num1 >= num2) {
                int times = num1 / num2;
                num1 -= num2 * times;
                res += times;
            } else {
                int times = num2 / num1;
                num2 -= num1 * times;
                res += times;
            }
        }
        return res;
    }
};
```

Time Complexity : $O(log(max(num1, num2)))$  
Space Complexity : $O(1)$  

```cpp=
class Solution {
public:
    int countOperations(int num1, int num2) {
        int res = 0;  // total number of subtraction operations
        while (num1 && num2) {
            // each step of the Euclidean algorithm
            res += num1 / num2;
            num1 %= num2;
            swap(num1, num2);
        }
        return res;
    }
};
```

Time Complexity : $O(log(max(num1, num2)))$  
Space Complexity : $O(1)$
