# [2154. Keep Multiplying Found Values by Two](https://leetcode.com/problems/keep-multiplying-found-values-by-two/description/)  

+ 日期 : 2025/11/19  

## 問題描述  

給定一個數字陣列 `nums` 和一個整數 `original`，  
如果 `original` 存在於 `nums` 中，則將 `original` 乘以 2，  
重複此過程直到 `original` 不再存在於 `nums` 中，回傳最終的 `original` 值。  

## 直覺想法  

1. `hash set`  
2. `while loop`  

## 題解  

```cpp=
class Solution {
public:
    int findFinalValue(vector<int>& nums, int original) {
        unordered_set<int> m_set;
        for(int num : nums)
            m_set.insert(num);
        while(m_set.count(original)) {
            original *= 2;
        }
        return original;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
