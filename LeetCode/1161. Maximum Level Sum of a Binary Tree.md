# [1161. Maximum Level Sum of a Binary Tree](https://leetcode.com/problems/maximum-level-sum-of-a-binary-tree/description/)  

+ 日期 : 2026/01/06  

## 問題描述  

找出 `tree` 各 `level` 的 `sum`，並回傳 `sum` 最大的 `level` (從 `1` 開始計算)。  

## 直覺想法  

1. BFS  

## 題解  

```cpp=
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    int maxLevelSum(TreeNode* root) {
        queue<pair<TreeNode *, int>> q;
        unordered_map<int, long> cnt;
        q.push({root, 1});
        while(!q.empty()) {
            auto [node, level] = q.front();
            q.pop();
            cnt[level] += node->val;
            if(node->left) {
                q.push({node->left, level + 1});
            }
            if(node->right) {
                q.push({node->right, level + 1});
            }
        }
        int res = 1e5;
        long curM = LONG_MIN;
        for(auto [level, val] : cnt) {
            if(val == curM) {
                res = min(res, level);
            } else if(val > curM) {
                res = level;
                curM = val;
            }
        }
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
