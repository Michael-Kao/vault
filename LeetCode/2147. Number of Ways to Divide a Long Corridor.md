# [2147. Number of Ways to Divide a Long Corridor](https://leetcode.com/problems/number-of-ways-to-divide-a-long-corridor/description/)  

+ 日期 : 2025/12/14  

## 問題描述  

給定一個字串 `corridor`，由 `S` 和 `P` 組成，請試著將 `corridor` 切開，使得每個區間內都有包含兩個 `S`，  
並且區間之間可以有任意數量的 `P`。請回傳有多少種切割方式，答案需要對 `10^9 + 7` 取模。  

## 直覺想法  

1. 計算目前 `S` 的數量  
2. 若 `S` 不為 `0` 且能被 `2` 整除，表示目前這個 `index` 可以進行切割，`cur + 1`  
3. 如果 `S` 數量為奇數或 `0`，表示目前還在等待第二個 `S` 加入當前區間，`cur` 不變  
4. 若 `S` 為奇數，但 `cur` 不為 `0`，表示前一個區間可切割數量為 `cur`，將 `res` 乘上 `cur`，並對 `10^9 + 7` 取模  
5. 最後判斷 `S` 數量是否為 `0` 或奇數，若是則回傳 `0`，否則回傳 `res`  

## 題解  

```cpp=
class Solution {
public:
    int numberOfWays(string corridor) {
        int n = corridor.size();
        int scnt = 0;
        int cur = 0;
        int res = 1;
        int MOD = 1e9 + 7;
        for(int i = 0; i < n; ++i) {
            char c = corridor[i];
            if(c == 'S')
                scnt += 1;
            if(scnt % 2 == 0 && scnt > 1) {
                cur += 1;
            } else {
                if(cur != 0)
                    res = (1ll * res * cur) % MOD;
                cur = 0;
            }

        }
        if(scnt == 0 || scnt & 1)    return 0;
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(1)$
