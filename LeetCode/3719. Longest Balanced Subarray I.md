# [3719. Longest Balanced Subarray I](https://leetcode.com/problems/longest-balanced-subarray-i/description/)  

+ 日期 : 2026/02/10  

## 問題描述  

Don't overthink it. There's only 1500 numbers in the input array.  
Brute force is fine.  

## 直覺想法  

## 題解  

```cpp=
class Solution {
public:
    int longestBalanced(vector<int>& nums) {
        int n = nums.size();
        int res = 0;
        for(int i = 0; i < n; ++i) {
            unordered_set<int> even, odd;
            for(int j = i; j < n; ++j) {
                if(nums[j] % 2 == 0)
                    even.insert(nums[j]);
                else
                    odd.insert(nums[j]);
                if(even.size() == odd.size()) 
                    res = max(res, j-i+1);
            }
        }
        return res;
    }
};
```

Time Complexity : $O(n^2)$  
Space Complexity : $O(n)$
