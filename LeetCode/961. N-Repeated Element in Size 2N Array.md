## 題解  

```cpp=
class Solution {
public:
    int repeatedNTimes(vector<int>& nums) {
        int n = nums.size() / 2;
        unordered_set<int> seen;
        for(int num : nums) {
            if(seen.count(num)) return num;
            seen.insert(num);
        }
        return 0;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
