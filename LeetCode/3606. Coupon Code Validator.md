# [3606. Coupon Code Validator](https://leetcode.com/problems/coupon-code-validator/description/)  

+ 日期 : 2025/12/13  

## 問題描述  

Too many stuff  

## 直覺想法  

1. 將所有合法的 `code`, `businessLine` 組成一個陣列  
2. 將該陣列進行排序  
3. 排序後陣列中的 `code` 即為結果  

## 題解  

```cpp=
class cmp {
public:
    bool operator()(vector<string> &a, vector<string> &b) {
        if(a[1] == b[1])
            return a[0] < b[0];
        return a[1][0] < b[1][0];
    }
};
class Solution {
public:
    vector<string> validateCoupons(vector<string>& code, vector<string>& businessLine, vector<bool>& isActive) {
        int n = code.size();
        vector<vector<string>> arr;
        for(int i = 0; i < n; ++i) {
            if(isCodeValid(code[i]) && isBusValid(businessLine[i]) && isActive[i]) {
                arr.push_back({code[i], businessLine[i]});
            }
        }

        sort(arr.begin(), arr.end(), cmp());
        vector<string> res;
        for(vector<string> cur : arr) {
            res.push_back(cur[0]);
        }
        return res;
    }

    bool isBusValid(string &str) {
        return (str == "electronics") || (str == "grocery") 
            || (str == "pharmacy") || (str == "restaurant");
    }

    bool isCodeValid(string &str) {
        if(str.empty()) return false;
        for(char c : str) {
            if(!isalnum(c) && c != '_')
                return false;
        }
        return true;
    }
};
```

Time Complexity : $O(nlogn)$  
Space Complexity : $O(n)$
