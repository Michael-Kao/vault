# [799. Champagne Tower](https://leetcode.com/problems/champagne-tower/description/)  

+ 日期 : 2026/02/14  

## 問題描述  

## 直覺想法  

把帕斯卡三角形想成直角三角形這招很聰明，然後模擬完整過程  

## 題解  

```cpp=
class Solution {
public:
    double champagneTower(int poured, int query_row, int query_glass) {
        vector<vector<double>> res(102, vector<double>(102, 0));
        res[0][0] = poured;
        for(int i = 0; i < query_row+1; ++i) {
            for(int j = 0; j < query_glass+1; ++j) {
                double val = (res[i][j] - 1.0) / 2.0;
                if(val > 0) {
                    res[i+1][j] += val;
                    res[i+1][j+1] += val;
                }
            }
        }
        return min(1.0, res[query_row][query_glass]);
    }
};
```

Time Complexity : $O(n^2)$  
Space Complexity : $O(n^2)$
