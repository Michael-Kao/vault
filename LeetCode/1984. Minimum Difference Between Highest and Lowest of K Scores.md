# [1984. Minimum Difference Between Highest and Lowest of K Scores](https://leetcode.com/problems/minimum-difference-between-highest-and-lowest-of-k-scores/description/)  

+ 日期 : 2026/01/25  

## 問題描述  

找出 `k` 個分數中，最高與最低的差值的最小值。  

## 直覺想法  

## 題解  

```cpp=
class Solution {
public:
    int minimumDifference(vector<int>& nums, int k) {
        sort(nums.begin(), nums.end());
        int l = 0, r = 0;
        int res = INT_MAX;
        while(r < nums.size()) {
            if(r - l + 1 > k) {
                l += 1;
            }
            if(r - l + 1 == k) {
                res = min(res, nums[r] - nums[l]);
            } 
            r += 1;
        }
        return res;
    }
};
```

Time Complexity : $O(nlgon)$  
Space Complexity : $O(1)$
