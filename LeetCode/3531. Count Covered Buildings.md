# [3531. Count Covered Buildings](https://leetcode.com/problems/count-covered-buildings/description/)  

+ 日期 : 2025/12/11  

## 問題描述  

給定一個整數 `n`，表示在座標平面 `n x n` 的範圍內，其中該範圍內的建築物由二維矩陣 `buildings` 表示，  
若有一個建築物被覆蓋，表示在建築物的上下左右四個方向都有其他建築物存在，則該建築物被視為覆蓋建築物，  
問有多少建築物是被覆蓋的？  

## 直覺想法  

1. 找出所有建築物的 `row` 和 `col` 中，最大和最小的座標  
2. 查看建築物的位置是否在最大和最小的範圍內，若是則表示該建築物被覆蓋  

## 題解  

```cpp=
class Solution {
public:
    int countCoveredBuildings(int n, vector<vector<int>>& buildings) {
        unordered_map<int, pair<int, int>> row;
        unordered_map<int, pair<int, int>> col;

        int size = buildings.size();
        for(int i = 0; i < size; ++i) {
            int r = buildings[i][0];
            int c = buildings[i][1];
            if(!row.count(r))
                row[r] = {INT_MAX, INT_MIN};
            if(!col.count(c))
                col[c] = {INT_MAX, INT_MIN};

            row[r].first = min(row[r].first, c);
            row[r].second = max(row[r].second, c);
            col[c].first = min(col[c].first, r);
            col[c].second = max(col[c].second, r);
        }

        int res = 0;
        for(int i = 0; i < size; ++i) {
            int r = buildings[i][0];
            int c = buildings[i][1];
            
            if(row[r].first < c && c < row[r].second &&
                col[c].first < r && r < col[c].second)
                res += 1;
        }
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
