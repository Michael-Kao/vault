# [3379. Transformed Array](https://leetcode.com/problems/transformed-array/description/)

## 題解  

```cpp=
class Solution {
public:
    vector<int> constructTransformedArray(vector<int>& nums) {
        int n = nums.size();
        vector<int> res(n);
        for(int i = 0; i < n; ++i) {
            int idx = (i + nums[i]) % n;
            idx = (idx + n) % n;
            res[i] = nums[idx];
        }
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(1)$
