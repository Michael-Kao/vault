# [67. Add Binary](https://leetcode.com/problems/add-binary/description/)  

+ 日期 : 2026/02/15  

## 問題描述  

給定兩個二進位字串，相加兩字串後回傳  

## 直覺想法  

模擬即可  

## 題解  

```cpp=
class Solution {
public:
    string addBinary(string a, string b) {
        string res = "";
        bool carry = false;
        int i = a.size()-1;
        int j = b.size()-1;
        while(i >= 0 && j >= 0) {
            int cnt = 0;
            if(a[i] == '1')
                cnt += 1;
            if(b[j] == '1')
                cnt += 1;
            if(carry)
                cnt += 1;
            
            res += (cnt == 1 || cnt == 3) ? '1' : '0';
            carry = (cnt > 1) ? 1 : 0;

            i -= 1;
            j -= 1;
        }
        while(i >= 0) {
            int cnt = 0;
            if(a[i] == '1')
                cnt += 1;
            if(carry)
                cnt += 1;
            
            res += (cnt == 1) ? '1' : '0';
            carry = (cnt > 1) ? 1 : 0;

            i -= 1;
        }
        while(j >= 0) {
            int cnt = 0;
            if(b[j] == '1')
                cnt += 1;
            if(carry)
                cnt += 1;
            
            res += (cnt == 1 || cnt == 3) ? '1' : '0';
            carry = (cnt > 1) ? 1 : 0;

            j -= 1;
        }
        if(carry)   res += '1';
        reverse(res.begin(), res.end());
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
