# [1653. Minimum Deletions to Make String Balanced](https://leetcode.com/problems/minimum-deletions-to-make-string-balanced/description/)  

+ 日期 : 2026/02/07  

## 問題描述  

給定一個字串 `s`，由小寫字母 `a` 和 `b` 組成，字串平衡表示所有 `a` 都在 `b` 的左邊，  
請問最少需要刪除多少字元才能讓字串平衡。

## 直覺想法  

**This problem has a really great editorial**  

1. 找出在 `index i` 之後的所有 `a` 數量  
2. 從 `0` 再次遍歷，這次計算在 `index i` 之前的 `b` 數量，並且加上 `index i` 之後的 `a` 數量，取最小值。  

## 題解  

```cpp=
class Solution {
public:
    int minimumDeletions(string s) {
        int n = s.size();
        vector<int> cnt(n, 0);
        for(int i = n-2; i >= 0; --i) {
            if(s[i+1] == 'a')   cnt[i] += 1;
            cnt[i] += cnt[i+1];
        }
        int pre = 0;
        for(int i = 1; i < n; ++i) {
            if(s[i-1] == 'b') {
                pre += 1;
            }
            cnt[i] += pre;
        }
        int res = INT_MAX;
        for(int i = 0; i < n; ++i) {
            res = min(res, cnt[i]);
        }
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
