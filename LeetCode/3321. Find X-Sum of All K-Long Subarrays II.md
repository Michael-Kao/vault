# [3321. Find X-Sum of All K-Long Subarrays II](https://leetcode.com/problems/find-x-sum-of-all-k-long-subarrays-ii/description/)  

+ 日期 : 2025/11/05  

## 問題描述  

[[LeetCode/3318. Find X-Sum of All K-Long Subarrays I.md]]  

## 直覺想法  

參考 [295. Find Median from Data Stream](https://leetcode.com/problems/find-median-from-data-stream/description/)  

1. 依舊使用一個 `hash map` 紀錄每個元素的出現次數  
2. 這次不使用 `heap`，改使用兩個 `ordered set`，以出現次數為排序基準  
3. 透過 `two pointer` 來控制子陣列長度  
4. `ordered set` 分別存放目前 `x` 個出現次數最高的元素，另一個存放子陣列中剩下的元素  
5. 每次更新 `hash map` 時先將該元素從 `ordered set` 中移除，更新後再重新插入剩餘元素的 `ordered set` 中  
6. 當剩餘元素的 `ordered set` 不為空，且 `x` 元素的 `ordered set` 數量不足 `x`，  
   則從剩餘元素中取出出現次數最高的元素加入 `x` 元素的 `ordered set` 中  
7. 如果兩邊數量正常，則判斷目前 `x` 的最小和剩餘的最大是否要交換  
8. 上方都更新完成後就能將目前的總和加入答案中  

## 題解  

```cpp=
class Solution {
public:
    vector<long long> findXSum(vector<int>& nums, int k, int x) {
        int n = nums.size();
        // freq, num
        set<pair<long, long>> topx, remain;
        unordered_map<long long, long long> cnt;
        int l = 0;
        long long cur = 0;
        vector<long long> res;
        for(int i = 0; i < n; ++i) {
            if(topx.count({cnt[nums[i]], nums[i]})) {
                topx.erase({cnt[nums[i]], nums[i]});
                cur -= nums[i] * cnt[nums[i]];
            }
            if(remain.count({cnt[nums[i]], nums[i]}))
                remain.erase({cnt[nums[i]], nums[i]});
            cnt[nums[i]] += 1;
            remain.insert({cnt[nums[i]], nums[i]});
            if(i - l + 1 > k) {
                if(topx.count({cnt[nums[l]], nums[l]})) {
                    topx.erase({cnt[nums[l]], nums[l]});
                    cur -= nums[l] * cnt[nums[l]];
                }
                if(remain.count({cnt[nums[l]], nums[l]}))
                    remain.erase({cnt[nums[l]], nums[l]});
                cnt[nums[l]] -= 1;
                remain.insert({cnt[nums[l]], nums[l]});
                l += 1;
            }
            while(topx.size() < x && !remain.empty()) {
                auto top = *remain.rbegin();
                topx.insert(top);
                remain.erase(top);
                cur += top.first * top.second;
            }
            if(!topx.empty() && !remain.empty() && 
                ((*topx.begin()).first < (*remain.rbegin()).first)) {
                auto large = *remain.rbegin();
                auto small = *topx.begin();
                topx.erase(small);
                topx.insert(large);
                remain.erase(large);
                remain.insert(small);

                cur -= small.first * small.second;
                cur += large.first * large.second;
            }
            if(!topx.empty() && !remain.empty() && 
                ((*topx.begin()).first == (*remain.rbegin()).first) &&
                ((*topx.begin()).second < (*remain.rbegin()).second)
            ) {
                auto large = *remain.rbegin();
                auto small = *topx.begin();
                topx.erase(small);
                topx.insert(large);
                remain.erase(large);
                remain.insert(small);

                cur -= small.first * small.second;
                cur += large.first * large.second;
            }
            if(i - l + 1 == k)
                res.push_back(cur);
        }
        if(n - l < k)
            res.push_back(cur);
        return res;
    }
};
```

Time Complexity : $O(nlogk)$  
Space Complexity : $O(n)$
