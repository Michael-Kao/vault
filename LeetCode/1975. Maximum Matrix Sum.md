# [1975. Maximum Matrix Sum](https://leetcode.com/problems/maximum-matrix-sum/description/)  

+ 日期 : 2026/01/05  

## 問題描述  

把矩陣中的負數經過翻轉後求最大總和  

## 直覺想法  

1. 計算矩陣中所有數 `abs` 的總和  
2. 紀錄矩陣中負數出現次數以及最小的 `abs` 值  
3. 如果負數出現偶數，表示可以全部翻轉為正數，直接回傳總和  
4. 如果負數出現奇數，表示最後會有一個負數無法翻轉，則從總和中扣除兩倍的最小 `abs` 值  

## 題解  

```cpp=
class Solution {
public:
    long long maxMatrixSum(vector<vector<int>>& matrix) {
        int n = matrix.size();
        vector<int> neg;
        long long res = 0;
        int cnt = 0;
        int minV = INT_MAX;
        for(int i = 0; i < n; ++i) {
            for(int j = 0; j < n; ++j) {
                if(matrix[i][j] <= 0)
                    cnt += 1;

                res += abs(matrix[i][j]);
                minV = min(minV, abs(matrix[i][j]));
            }
        }
        if(cnt % 2 == 1)    res -= 2 * minV;
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(1)$
