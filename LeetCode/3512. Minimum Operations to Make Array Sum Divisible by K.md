# [3512. Minimum Operations to Make Array Sum Divisible by K](https://leetcode.com/problems/minimum-operations-to-make-array-sum-divisible-by-k/description/)  

+ 日期 : 2025/11/29  

## 問題描述  

給定整數陣列 `nums` 和整數 `k`，每次操作可以選 `nums[i] - 1`，  
問最少操作幾次，可以讓 `nums` 的總和可以被 `k` 整除。  

## 直覺想法  

1. 計算 `nums` 中所有數 `% k` 的總和 `total_mod`。  
2. 回傳 `total_mod`  

## 題解  

```cpp=
class Solution {
public:
    int minOperations(vector<int>& nums, int k) {
        int res = 0;
        for(int num : nums) {
            res = (res + (num % k)) % k;
        }
        return res;
    }
};
```

Time Compleixty : $O(n)$  
Space Compleixty : $O(1)$
