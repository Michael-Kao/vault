# [3075. Maximize Happiness of Selected Children](https://leetcode.com/problems/maximize-happiness-of-selected-children/description/)  

+ 日期 : 2025/12/25  

## 問題描述  

給定一個陣列 `happiness` 和整數 `k`，可以從陣列中取 `k` 個數，每當取出一個數後，其餘的 `happiness` 都會 `-1`，  
求最大化所取數的總和。  

## 直覺想法  

1. 排序  
2. 從最大開始取，每次取完用 `dec` 紀錄目前減少的值  
3. 每個值為 `max(0, happiness[i] - dec)`  

## 題解  

```cpp=
class Solution {
public:
    long long maximumHappinessSum(vector<int>& happiness, int k) {
        sort(happiness.begin(), happiness.end());
        int n = happiness.size();
        int dec = 0;
        long long res = 0;
        for(int i = n-1; i >= 0; --i) {
            res += max((happiness[i] - dec), 0);
            k -= 1;
            dec += 1;
            if(k == 0)  break;
        }
        return res;
    }
};
```

Time Complexity : $O(nlogn)$  
Space Complexity : $O(1)$
