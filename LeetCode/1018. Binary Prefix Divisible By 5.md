# [1018. Binary Prefix Divisible By 5](https://leetcode.com/problems/binary-prefix-divisible-by-5/description/)  

+ 日期 : 2025/11/24  

## 問題描述  

給定一個由 `0` 和 `1` 組成的陣列 `nums`，由 `index = 0` 開始遍歷，  
將位元作為二進位數字來解讀，假設 `nums = [1, 0, 1]`，那麼各 `index` 對應的二進位數字為 `[1, 2, 5]`，  
問每個二進位數字是否能被 `5` 整除，回傳一個布林陣列。  

## 直覺想法  

由於 `nums` 的長度最多為 `10^4`，因此不能直接將二進位數字轉換成十進位數字來判斷是否能被 `5` 整除，  
為了避免 `overflow`，我們可以儲存當前數字對 `5` 取餘數的結果，而每次右移一位相當於乘以 `2`，  
因此步驟如下:  

1. 建立長度為 `n` 的布林陣列 `res`  
2. 遍歷 `nums` 陣列，並以 `cur` 變數儲存目前的數字  
3. 對於每個 `index i`，將先前的 `cur * 2`，並且加上 `nums[i]` 後對 `5` 取餘數，更新 `cur`  
4. 判斷 `cur` 是否等於 `0`，若是則表示能被 `5` 整除，將 `res[i]` 設為 `true`，否則設為 `false`  

## 題解  

```cpp=
class Solution {
public:
    vector<bool> prefixesDivBy5(vector<int>& nums) {
        int n = nums.size();
        vector<bool> res(n);
        int cur = 0;
        for(int i = 0; i < n; ++i) {
            cur = (cur * 2) % 5;
            cur = (cur + nums[i]) % 5;
            res[i] = (cur == 0);
        }
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
