# [3289. The Two Sneaky Numbers of Digitville](https://leetcode.com/problems/the-two-sneaky-numbers-of-digitville/description/)  

+ 日期 : 2025/10/31  

## 問題描述  

給定一個數字陣列 `nums`，長度為 `n+2`，裡面的數字從 `[0, n-1]`，  
會有兩個數字出現兩次，找出這兩個數字。  

## 直覺想法  

1. `hash map` 紀錄  

## 題解  

```cpp=
class Solution {
public:
    vector<int> getSneakyNumbers(vector<int>& nums) {
        int m = nums.size();
        vector<bool> seen(m-2, false);
        int idx = 0;
        vector<int> res(2, -1);
        for(int i = 0; i < m; ++i) {
            if(seen[nums[i]]) {
                res[idx++] = nums[i];
            }
            if(idx == 2)    return res;
            seen[nums[i]] = true;
        }
        return {};
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
