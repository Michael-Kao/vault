# [1523. Count Odd Numbers in an Interval Range](https://leetcode.com/problems/count-odd-numbers-in-an-interval-range/description/)  

+ 日期 : 2025/12/07  

## 問題描述  

給定兩個整數 `low` 和 `high`，找出區間 `[low, high]` 有多少個奇數。  

## 直覺想法  

1. 該區間數字有 `high - low + 1` 個  
2. 若 `low` 和 `high` 同為奇數，表示區間起始被位移到奇數位置，則整體奇數個數會多一個  
3. 否則奇數數量為 `(high - low + 1) // 2`  

## 題解  

```cpp=
class Solution {
public:
    int countOdds(int low, int high) {
        int res = (high - low + 1) / 2;
        return ((low & 1) && (high & 1)) ? res + 1 : res;
    }
};
```

Time Complexity : $O(1)$  
Space Complexity : $O(1)$
