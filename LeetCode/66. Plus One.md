# [66. Plus One](https://leetcode.com/problems/plus-one/description/)  

+ 日期 : 2026/01/01  

## 問題描述  

給一個數字陣列當作一個 `digit` 的表示，對這個數字加一，並回傳新的陣列。  

## 直覺想法  

1. 從最小位開始建立  
2. 完成後反轉  

## 題解  

```cpp=
class Solution {
public:
    vector<int> plusOne(vector<int>& digits) {
        int n = digits.size();
        vector<int> res(n, 0);

        int plus = 1;
        for(int i = n-1; i >= 0; --i) {
            if(digits[i] + plus == 10) {
                res[n-i-1] = 0;
            } else {
                res[n-i-1] = digits[i] + plus;
                plus = 0;
            }
        }
        if(plus)    res.push_back(1);
        reverse(res.begin(), res.end());
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
