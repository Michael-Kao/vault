# [1680. Concatenation of Consecutive Binary Numbers](https://leetcode.com/problems/concatenation-of-consecutive-binary-numbers/description/)  

+ 日期 : 2026/02/28  

## 問題描述  

將數字 `1` 到 `n` 以二進位形式表示，並且將其串接在一起，  
問最後串接出來的數字對 `10^9 + 7` 取餘數後的結果為多少。  

## 直覺想法  

如果將數字真的轉成二進位字串，會無法正確處理 `int` 型態的值，  
因此必須使用 `int` 並以 `long long` 輔助完成。  

1. 從 `1` 到 `n`，每個數字找出其二進位的長度  
2. 將目前總數左移 `length` 位，並以 `|` 將目前數字更新  
3. 每次更新後都對 `10^9 + 7` 取餘數，避免數字過大無法處理  

## 題解  

```cpp=
class Solution {
public:
    int concatenatedBinary(int n) {
        int MOD = 1e9 + 7;
        long long cur = 0;
        for(int i = 1; i <= n; ++i) {
            int shift = 0;
            int num = i;
            while(num > 0) {
                shift += 1;
                num >>= 1;
            }
            cur = (cur << shift) | i;
            cur = (cur % MOD);
        }
        return cur % MOD;
    }
};
```

Time Complexity : $O(32n)$  
Space Complexity : $O(1)$
