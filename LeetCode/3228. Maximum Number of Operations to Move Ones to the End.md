# [3228. Maximum Number of Operations to Move Ones to the End](https://leetcode.com/problems/maximum-number-of-operations-to-move-ones-to-the-end/description/)  

+ 日期 : 2025/11/13  

## 問題描述  

給定一個字串，該字串由 `0` 和 `1` 組成，可以針對字串做以下兩種操作：  

1. 將字串最右方的 `1` 移到字串最結尾  
2. 若字元 `1` 不是字串中最後的 `1`，則下一個字串中的 `1` 作為移動的終點  

問最多幾次操作，能將所有的 `1` 移到字串的最右方？  

## 直覺想法  

1. 若目前字元為 `1`，計算 `i - pre > 1`，若 `i-pre = 1`，表示相鄰，不需要操作  
2. 若不為 `i - pre > 1`，則在 `i` 之前的所有 `1` 都可以右移到 `i-1` 的位置，操作次數為在 `i` 之前的 `1` 的數量  
3. 更新 `pre` 以及目前 `1` 的數量  

## 題解  

```cpp=
class Solution {
public:
    int maxOperations(string s) {
        s += '1';
        int n = s.size();
        int res = 0;
        int pre = -1;
        int cnt = 0;
        for(int i = 0; i < n; ++i) {
            if(s[i] == '1') {
                if(i - pre > 1) {
                    res += cnt;
                }

                cnt += 1;
                pre = i;
            }
        }
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(1)$
