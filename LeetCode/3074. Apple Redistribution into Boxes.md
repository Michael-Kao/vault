## 題解  

```cpp=
class Solution {
public:
    int minimumBoxes(vector<int>& apple, vector<int>& capacity) {
        int n = apple.size();
        int m =capacity.size();
        int total = 0;
        for(int num : apple) {
            total += num;
        }
        sort(capacity.begin(), capacity.end());
        int res = 0;
        for(int i = m - 1; i >= 0; --i) {
            if(total <= 0)  break;
            res += 1;
            total -= capacity[i];
        }
        return res;
    }
};
```

Time Complexity : $O(n + mlogm)$  
Space Complexity : $O(1)$
