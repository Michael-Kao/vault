# [2483. Minimum Penalty for a Shop](https://leetcode.com/problems/minimum-penalty-for-a-shop/description/)  

+ 日期 : 2025/12/26  

## 問題描述  

給定一個字串 `customers`，由 `Y` 和 `N` 組成，表示每個小時是否有顧客來店裡，  
若開店時有 `N`，則處罰 `+1`，若提前關店時有 `Y`，則處罰 `+1`，  
問在哪個時間點關店，可以讓處罰最小？  

## 直覺想法  

1. 利用 `prefix sum` 紀錄目前時間點之前有多少個 `Y`  
2. 每個時間點的處罰為 `penalty = (total_Y - prefix_Y) + prefix_N`  
3. 找出最小 `penalty` 且最小的時間點  

## 題解  

```cpp=
class Solution {
public:
    int bestClosingTime(string customers) {
        int n = customers.size();
        vector<int> prefix(n+1);
        int cnt = 0;
        for(int i = 0; i < n; ++i) {
            prefix[i] = cnt;
            if(customers[i] == 'Y') {
                cnt += 1;
            }
        }
        prefix[n] = cnt;
        int res = 0;
        for(int i = 1; i <= n; ++i) {
            int penalty = (i - prefix[i]) + (prefix[n] - prefix[i]);
            int pre_pen = (res - prefix[res]) + (prefix[n] - prefix[res]);
            if(penalty < pre_pen) {
                res = i;
            }
        }
        return res;
    }
};
```

Time Complexity : $O(n)$  
Space Complexity : $O(n)$
